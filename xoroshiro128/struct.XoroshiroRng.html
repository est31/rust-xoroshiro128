fn new() -> Result<Self>

Creates a new XoroshiroRng instance which is randomly seeded.

Errors

When created this way the initial values of this Rng are seeded using an OsRng. If this fails then this method will pass through the error from OsRng.

Examples

use xoroshiro128::XoroshiroRng;
use rand::Rng;

let rng = XoroshiroRng::new();
let x: u32 = rng.unwrap().gen();
println!("{}", x);Run

fn new_unseeded() -> Self

Creates a new XoroshiroRng instance which is not seeded.

The initial values of this Rng are constants, so all generators created by this function will yield the same stream of random numbers. It is highly recommended that this is created through SeedableRng instead of this function.
